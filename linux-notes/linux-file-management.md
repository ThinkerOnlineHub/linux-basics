# Linux File Management Practice

## Objective
Practice Linux file creation and file handling operations.

---

## Commands Used

```bash
touch think
touch think2
cat filename
vim filename
mkdir ops-practice
```

---

## Key Learnings

- Created files using `touch`.
- Edited files using `vim` editor.
- Viewed file contents using `cat`.
- Created directories for organizing practice files.

---

## Why This Matters

Linux file management is important for:

- Server administration
- Configuration management
- Shell scripting
- DevOps automation
- Log management
