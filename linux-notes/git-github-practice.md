# Git & GitHub Practice on Linux

## Objective
Practice Git and GitHub workflows from Linux terminal.

---

## Commands Used

```bash
git remote add origin https://github.com/ThinkerOnlineHub/git-github-mastery/
git push -u origin main
git pull
```

---

## Key Learnings

- Connected local repository with GitHub remote repository.
- Practiced Git push and pull workflows.
- Understood Linux-based Git operations.

---

## Why This Matters

Git and GitHub are important for:

- DevOps workflows
- Version control
- Infrastructure documentation
- Collaboration
- CI/CD pipelines
