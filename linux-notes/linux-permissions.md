# Linux File Permissions Practice

## Objective
Understand executable permissions and shell script execution in Linux.

---

## Commands Used

```bash
chmod +x Loop_file-check.sh
./Loop_file-check.sh
```

---

## Key Learnings

- Used `chmod +x` to make shell scripts executable.
- Executed shell scripts directly from terminal.
- Learned Linux executable permission concepts.

---

## Real-World Usage

Linux permissions are critical for:

- Server security
- Script execution
- DevOps automation
- CI/CD pipelines
- Infrastructure operations
