# Linux Basic Navigation Commands Practice

## Objective
Practice basic Linux navigation and directory management commands on AWS EC2 Ubuntu instance.

---

## Commands Practiced

```bash
pwd
ls
ls -la
cd ~
cd /
ls /usr/
ls /usr/bin/
ls /bin
mkdir ~/ops-practice
```

---

## Key Learnings

- Used `pwd` to identify current working directory.
- Explored files and hidden files using `ls -la`.
- Practiced navigation between home and root directories.
- Explored Linux system binaries under `/usr/bin` and `/bin`.
- Created practice directories for Linux learning.

---

## Real-World Importance

These commands are fundamental for:

- Linux administration
- Server troubleshooting
- Infrastructure support
- Cloud operations
- DevOps workflows
