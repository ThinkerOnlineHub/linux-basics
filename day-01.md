# Day 01 - Linux Basics
## Commands Practiced
- pwd : Function: Prints the current working directory.
- ls : Function: Lists files and directories in the current directory.
### Common Options:
  ls -l → Detailed list with permissions, size, date.
  ls -a → Includes hidden files.
- cd : Function: Changes the current working directory.
- mkdir : Function: Creates a new directory.
- touch : Function: Creates an empty file or updates the timestamp of an existing file.
- rm : Function: Removes (deletes) files or directories.
  ### Common Options:
  rm -r folder → Deletes a directory and its contents.
  rm -f file.txt → Force delete without confirmation.

  | **Command** | **Function**                                      | **Example Usage**                |
|-------------|----------------------------------------------------|-----------------------------------|
| `pwd`       | Prints the current working directory              | `pwd` → `/home/user/Documents`   |
| `ls`        | Lists files and directories in the current folder | `ls -l` → Detailed list          |
| `cd`        | Changes the current working directory             | `cd /home/user/Desktop`          |
| `mkdir`     | Creates a new directory                           | `mkdir new_folder`               |
| `touch`     | Creates an empty file or updates timestamp        | `touch file.txt`                 |
| `rm`        | Removes files or directories                      | `rm file.txt` or `rm -r folder`  |
