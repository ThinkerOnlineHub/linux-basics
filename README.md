## 🧠 Skills Covered (Week 1 Completed ✅)

### 🔹 Core Linux Fundamentals

* Command line navigation
* File system structure
* File operations

### 🔹 Security & Access Control

* File permissions (`chmod`)
* Ownership (`chown`)
* Users, groups & sudo access

### 🔹 System Operations

* Process monitoring (`ps`, `top`)
* Service management (`systemctl`)
* Process control & troubleshooting

### 🔹 Networking Basics

* Connectivity checks (`ping`, `curl`)
* Port & socket visibility (`ss`, `netstat`)
* Basic network troubleshooting

### 🔹 Disk & Filesystem

* Disk usage analysis (`df`, `du`)
* Mounting & filesystem basics

### 🔹 Logs & Debugging

* System logs (`journalctl`)
* Real-world troubleshooting approach
